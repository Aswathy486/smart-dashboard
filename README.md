# Personal Productivity Dashboard

A responsive web application built using HTML, CSS, and JavaScript that helps users manage tasks and view real-time weather data.

---

## 🚀 Features

- ⏰ Dynamic greeting based on time  
- 📝 Task management system:
  - Add tasks  
  - Edit tasks  
  - Mark tasks as completed  
  - Delete individual tasks  
  - Clear all tasks  
- 💾 Persistent storage using localStorage  
- 🌦️ Live weather search using OpenWeather API  
- ⚡ Loading state and error handling for API  
- 📱 Responsive UI design  

---

## 🛠️ Tech Stack

- HTML  
- CSS (Flexbox, responsive design)  
- JavaScript (DOM manipulation, event handling)  
- OpenWeather API  
- localStorage
